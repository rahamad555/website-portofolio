# 💻 Penjelasan Setiap Code Section


# Head




<img width="812" height="265" alt="image" src="https://github.com/user-attachments/assets/62db650f-33fb-49b4-91f2-5e640aaf946b" />









Bagian section Head ini berfungsi untuk mengatur identitas dan pengaturan dasar halaman website. Pada bagian ini terdapat pengaturan karakter dan tampilan responsif agar website dapat menyesuaikan berbagai ukuran layar. Tag <title> digunakan untuk menentukan judul halaman yaitu “Portfolio Ramadhan” yang muncul pada tab browser. Selain itu terdapat penghubung file CSS melalui yang digunakan untuk memuat styling dan desain tampilan website.



# Navbar





<img width="669" height="233" alt="image" src="https://github.com/user-attachments/assets/1d0f0cba-7995-4f24-bb04-b5bdf8d29550" />








Bagian section Navbar ini berfungsi sebagai menu navigasi utama pada website yang memudahkan pengguna berpindah antar section. Di dalamnya ada elemen logo berupa nama “MyPortfolio” sebagai identitas website, serta daftar menu navigasi yaitu Home, About, dan Certificates. Setiap menu menggunakan link anchor (#id) yang terhubung langsung ke section terkait pada halaman sehingga ketika diklik halaman akan berpindah ke bagian tersebut.
